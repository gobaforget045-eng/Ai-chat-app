# AI Chat Application - ChatGPT Style APK

A feature-rich Android chat application with AI integration, inspired by ChatGPT.

## Features

✨ **Core Features:**
- Real-time AI conversations
- Chat history management
- Multiple conversation threads
- Message search functionality
- Voice input support
- Copy/share messages
- Regenerate responses
- Clear conversations

🎨 **UI/UX:**
- Modern Material Design 3
- Dark/Light theme toggle
- Smooth animations
- Responsive layout
- Bottom sheet menus

🔒 **Security:**
- Local SQLite database
- Encrypted API key storage
- Session management

🚀 **Advanced Features:**
- Temperature/Creativity slider
- Model selection (GPT-3.5, GPT-4)
- System prompts customization
- Token usage display
- Response streaming

## Setup Instructions

1. Clone the repository
2. Open in Android Studio
3. Add your OpenAI API key in Settings
4. Build and run

## Technologies Used

- Kotlin
- Material Design 3
- Room Database
- Retrofit + OkHttp
- MVVM Architecture
- Hilt Dependency Injection
- Coroutines
